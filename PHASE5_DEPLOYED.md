### 2. Tool Integration

- Handle tool errors gracefully

### 3. API Endpoints
Available endpoints:
- `GET /api/mcp` - List all servers
- `POST /api/mcp/register` - Register new server

- `PUT /api/mcp/[id]` - Update server
- `DELETE /api/mcp/[id]` - Remove server
- `POST /api/mcp/[id]/toggle` - Enable/disable
