🚀 LANGKAH SELANJUTNYA: DEPLOY KE VERCEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Total waktu: ~10 menit

STEP 1: Buka Vercel (1 menit)
┌─────────────────────────────────────────────────────────────────┐
│ 1. https://vercel.com                                           │
│ 2. Login dengan GitHub                                          │
│ 3. Klik "New Project"                                           │
└─────────────────────────────────────────────────────────────────┘

STEP 2: Import Project (1 menit)
┌─────────────────────────────────────────────────────────────────┐
│ 1. Search: "Project-Zulu"                                       │
│ 2. Atau: mnhidayatgani/Project-Zulu                             │
│ 3. Klik "Import"                                                │
└─────────────────────────────────────────────────────────────────┘

STEP 3: Configure (1 menit)
┌─────────────────────────────────────────────────────────────────┐
│ Framework:  Next.js (auto-detected) ✓                           │
│ Build:      npm run build (default) ✓                           │
│ Output:     .next (default) ✓                                   │
│                                                                 │
│ TIDAK PERLU UBAH APAPUN!                                        │
└─────────────────────────────────────────────────────────────────┘

STEP 4: Environment Variables (5 menit)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│ Klik tab "Environment Variables"                                │
│                                                                 │
│ Copy dari .env.local Anda dan paste:                            │
│                                                                 │
│ 🔴 REQUIRED (5 variables):                                      │
│   • NEXT_PUBLIC_SUPABASE_URL                                    │
│   • NEXT_PUBLIC_SUPABASE_ANON_KEY                               │
│   • SUPABASE_SERVICE_ROLE_KEY                                   │
│   • OPENAI_API_KEY                                              │
│   • CSRF_SECRET                                                 │
│                                                                 │
│ 🟡 OPTIONAL (AI providers lain):                                │
│   • ANTHROPIC_API_KEY (Claude)                                  │
│   • GOOGLE_GENERATIVE_AI_API_KEY (Gemini)                       │
│   • MISTRAL_API_KEY                                             │
│   • XAI_API_KEY (Grok)                                          │
│   • EXA_API_KEY (Web search)                                    │
│   • GITHUB_TOKEN (Code search)                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

STEP 5: DEPLOY! (2-3 menit)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│ 1. Klik tombol "Deploy"                                         │
│ 2. Tunggu build selesai (2-3 menit)                             │
│ 3. DONE! 🎉                                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
